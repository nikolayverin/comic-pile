Comic Pile

Portable release notes

1. Launch `Comic Pile.exe`.
2. The app uses the `Database` folder next to the app as its starter library.
3. The first release includes a small demo library so you can browse and read immediately.
4. During normal app use, Comic Pile can create runtime state inside `Database/.runtime`.

Bundled public-domain demo comics:
- Courtesy of ComicBookPlus.Com

Bug reports:
- GitHub Issues will be the public bug-report path once the repository issue flow is opened for users.

Useful report details:
- what happened
- what you expected
- how to reproduce it
- your Windows version
- your Comic Pile version from About
